# fish-fzf-git-recent-branch

[![Build Status][travis-badge]][travis-link]
[![Slack Room][slack-badge]][slack-link]

Efficient fish keybinding for fzf with git branch

## Install

With [fisherman]

```
fisher tsub/fish-fzf-git-recent-branch
```

## Usage

```fish
fish-fzf-git-recent-branch
```
